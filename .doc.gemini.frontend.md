# VPoR Frontend Specifications

## 1. Overview
The Frontend is the "Glass Vault" interface. It must be clean, futuristic, and reassuring.

**Tech Stack:**
-   **Framework:** Next.js
-   **Styling:** Tailwind CSS (Glassmorphism theme)
-   **Web3:** RainbowKit + Wagmi + Viem

## 2. User Flows

### Flow A: Public Verification (The "Solvency Check")
*Anyone can see this.*
1.  **Landing Page:** Big bold numbers.
    -   **Total Assets:** $1,000,000 (Fetched from Chainlink Event)
    -   **Total Liabilities:** $950,000 (Fetched from Liability Root)
    -   **Reserve Ratio:** 105% (Green)
2.  **Visuals:** A live graph showing the updates coming in from the VPoR Chain.

### Flow B: Private Verification (The "My Money" Check)
*Only the user can see this.*
1.  **Connect Wallet:** User connects Metamask.
2.  **Scan Events:** The frontend queries the VPoR Chain for `ProofOfInclusion` events indexed by the user's address.
3.  **Decrypt:**
    -   Found an event? Prompt user to decrypt.
    -   **Metamask Call:** `window.ethereum.request({ method: 'eth_decrypt', params: [encryptedMessage, account] })`
4.  **Verify:**
    -   Display the decrypted data: "Balance: 5.5 ETH".
    -   Re-calculate the Merkle Path locally to prove it matches the on-chain Root.
    -   **Result:** "✅ Verified. Your funds are included in Block #102."

## 3. UI Components

### The "Glass Vault" Card
A central card component with a frosted glass effect (`backdrop-blur-md bg-white/10`) containing the core metrics.

### The "Sync" Indicator
A status light showing the health of the system.
-   **Green:** Synced (Assets & Liabilities updated < 1 hour ago).
-   **Yellow:** Stale (Updates > 1 hour).
-   **Red:** Mismatch (Liabilities > Assets).

## 4. Hackathon Demo Mode
To ensure the demo goes smoothly:
-   **Pre-load Keys:** Have a "Demo Wallet" button that auto-fills a private key for a test user, so judges don't need to install Metamask if they don't want to.
-   **Mock Data:** If the Chainlink testnet is flaky, have a "Mock Mode" switch (hidden) to use hardcoded asset data.
